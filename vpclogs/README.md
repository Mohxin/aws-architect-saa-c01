# vpc logs

- monitor networking in  vpc
- streamed to cloudwatch
- configured in cloudwatch

## vpc flow logs

- monitor ip traffic
- set to either
  - vpc
  - subnet
  - nic
- set to monitor
  - accepted traffic
  - rejected traffic
  - all traffic

## vpc access logs

- delivered to s3
- monitor examples
  - elb traffic
  - cloudfront traffic
  - s3 traffic
