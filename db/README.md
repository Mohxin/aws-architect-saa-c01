# dbs

## managed database services
### amazon rds 
- relational database
- can run the following:
    - sql server
    - mysql
    - postgres
    - mariadb
    - oracledb
    - orora (sp)

### amazon redshift
- datawarehouse 

### dynamodb 
- document database

### amazon elasicache
- key-value database
- ca run the following:
    - redis
    - memcache
  
### amazon neptune
- graph database

### elastic map reduce (emr)
- column database
- can run the following:
    - hbase
